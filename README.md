# NIT Rourkela Events Website
This is a  front-end project that displays information about events at *NIT Rourkela*.  
The website is created using only *HTML and CSS* to practice basic web development concepts.

## About the Project
The project focuses on understanding:
- HTML structure
- CSS styling
- Adding simple interactivity using CSS only

It is a static website with no backend or JavaScript.

## Features
- Simple and clean design
- Hover effects
- CSS transitions
- Button animations
- Toggle effects using CSS

## Technologies Used
- *HTML5*
- *CSS3*
